Quick Start

  1. 只需填写：API 接口地址（格式如：http://192.168.88.8:8000/openai/deepseek/v1/chat/completions）和 API 密钥 (API Key)；
  2. 点击密钥输入框右侧的刷新按钮，在下拉菜单中选择测试模型；
  3. 在参数配置中填写您希望测试的测试轮数 (Rounds)，然后点击 开始测试 即可；
  4. Max Tokens / Temperature / 单个测试超时 / 最大重试次数 均无须修改，系统已预设与官方默认规则保持一致；
  5. 任何时刻，你都可以点击热力图中的方块，查看当前题目的过程数据日志；
  6. 测试完成后，可以导出过程日志到本地保存。
